# Jeremy-s-resurrection-effect-project
Power BI dashboard analyzing ROI and profitability of major movie franchises, highlighting top performing movies and insights from budget vs box office data

## Overview
The Resurrection Effect Project is a data analytics dashboard built in Power BI to explore the financial performance of popular movie franchises. The goal was to analyze which movies and franchises deliver the highest ROI (Return on Investment) and profitability.

## Objective
- Identify the most profitable movies and franchises.  
- Calculate ROI for each movie based on box office revenue and budget.  
- Visualize insights in a clean, interactive dashboard.

## Franchises Analyzed
- Spiderman  
- Fast & Furious  
- Star Wars  
- Jurassic Park / World  

## Key Insights
- Spiderman had the *highest ROI* among the franchises.  
- Fast & Furious generated the *highest total profit*.  
- The analysis highlights how budget and box office performance affect franchise success.

## Tools Used
- *Microsoft Excel* – data cleaning and preparation  
- *Power BI* – dashboard visualization and analysis  

## How to Explore the Dashboard
1. Open the Resurrection_Effect_Project.pbix file in Power BI Desktop.  
2. Use the filters to view ROI and profitability by franchise or movie.  
3. Hover over visuals to see detailed metrics and insights.
